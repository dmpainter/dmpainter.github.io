Descriptive text here. You will adjust the width of this table data cell so that it aligns nicely with the image at the left.

All you need to know to do this is that the total width of this table is 900 pixels. The table uses 5 pixels of cellpadding and is two data cells wide - so there is 20 pixels of cell padding to account for. You will take 900 pixels (table width) - 20 pixels (cell padding) - xxx pixels (width of your photo) = the width of the descriptive text data cell.

The width setting is an attribute within the table datacell td code. It style="width:xxxpx" - you will change the xxx to the number of pixels wide that you would like this table data cell to be.
